# GuideCX CLI
GuideCX Web User Interface requires patience.

For developers riding in the fast lane, this CLI will get the job done without testing your patience.

## Usage

```shell
$ make build
$ GUIDECX_API_KEY="<your-api-key>" ./gcx-linux-amd64 time new
```
